# 🚀 Project Management System (SaaS)  
### 💻 MERN Stack | 🌐 Next.js | 🎨 Tailwind CSS | 🔒 Google OAuth 2.0 | 📊 Agile Workflow

A powerful multi-tenant SaaS platform designed to supercharge team productivity and streamline Agile project management. Built using the modern MERN stack, this app ensures **scalability**, **security**, and **speed** – everything you need for **collaborative task management** at scale!

---

## 🌟 Features

- 🏢 **Multi-Tenant Architecture**  
   Isolated **workspaces** with **Role-Based Access Control**  
   (`Owner / Admin / Member`)

- 🔐 **Authentication & Authorization**  
   Seamless login with **Google OAuth 2.0**, secure session handling with cookies

- ✅ **Task & Epic Management**  
   Fully-featured **CRUD operations** with:
   - 🎯 Priority & Status Filters  
   - 🔎 Search  
   - 📄 Pagination  
   - 📈 Analytics Dashboards

- ⚙️ **Robust Backend (Node.js + MongoDB)**  
   - 💾 **Mongoose Transactions** for data integrity  
   - 🔁 **Caching** for low-latency responses  
   - 🌱 Data seeding for test environments  
   - 🌐 Optimized APIs for high-traffic loads

- 💅 **Frontend** with **Next.js**, **React**, and **Tailwind CSS**  
   - Responsive & sleek UI  
   - Built with **TypeScript** for type safety
